Feature 1 — Student Registration
If students can't register on the home page
Students go directly to the quiz page — they are prompted for their name there. Registration is optional for the roster but not for the quiz.
Tell students: open this URL directly
profdrtan.github.io/research-methods/quiz.html
Enter your name when prompted and start the quiz
Feature 2 — Quiz Not Loading
If quiz.html shows blank or errors
Hard refresh first. If still broken, students can access the quiz on a different browser or device. The quiz has zero server dependencies — it runs entirely in the browser.
Ask students to press Ctrl+Shift+R (laptop) or close and reopen Chrome (phone)
If still broken — ask them to try incognito mode
Quiz works on any browser: Chrome, Safari, Edge, Firefox
Feature 3 — Leaderboard Not Showing Scores
If leaderboard.html shows "Could not load scores"
The leaderboard reads from the Cloudflare Worker. If Worker is down, scores are still saved locally on each student's device. Ask each student to read out their score verbally or show their screen.
Open leaderboard and check — scores usually load within 5 seconds
If not loading: ask students to open quiz results on their own screen
Backup — ask students verbally: "who got above 75%? above 50%?"
profdrtan.github.io/research-methods/leaderboard.html
Feature 4 — Roster Not Showing Students
If Class Roster shows 0 students
The roster pulls from the Cloudflare Worker. If Worker is slow, it falls back to your local browser's localStorage. This only affects your instructor view — students can still take the quiz.
Wait 10 seconds and refresh — Worker usually responds within 5s
If still empty — the quiz and leaderboard still work independently
Proceed with class — roster is cosmetic, not functional
Feature 5 — Speaker Notes Not Loading
If notes.html doesn't open on your phone
The speaker notes are a static HTML file — no server needed. If it doesn't load, it's a network issue on your phone.
Switch from WiFi to mobile data (or vice versa) and retry
Open in incognito tab to bypass cache
profdrtan.github.io/research-methods/notes.html
Feature 6 — Answer Scripts Not Loading
If answers.html is slow or blank
answers.html is large (145KB) and may be slow on mobile data. Open it before class starts so it caches on your device.
Open answers.html NOW before class — let it fully load and cache
Keep the tab open throughout the day — don't close it
profdrtan.github.io/research-methods/answers.html
